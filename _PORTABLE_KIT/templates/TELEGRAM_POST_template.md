# Шаблон Telegram поста

## Технические ограничения
- **Bot API caption** (фото + подпись): ≤ 1024 символа
- **Telegram Premium вручную**: ≤ 2048 символа (влезает большинство постов)
- **Обычный текст через бота**: без лимита (sendMessage)
- **Лучшая стратегия для длинного поста**: 2 сообщения (фото без caption + текст) или Premium вручную

---

## Шаблон: анонс релиза

```
[Заголовок одной строкой — что случилось] 🏁

[1-2 предложения: суть что сделал / выпустил]

---

**[Основной раздел 1:]**

[Текст раздела]

---

**[Основной раздел 2:]**

[Текст раздела]

---

**[Список/история изменений:]**

vX.X — [суть одной строкой]
vX.X — [суть одной строкой]

---

[Завершающий абзац — контекст, что это значит, эмоция]

---

🔗 **[Название ссылки]:** [URL]

[Что скачивать / куда идти]
```

---

## Правила тона
- Неформально, мат ок, не корпоративно
- Эмодзи органично, не в каждой строке
- Читатель знает контекст — без "если вы пропустили" объяснялок
- "Мы живём в будущем 🤖" — если пост про AI-разработку

## Про AI-разработку — блок
```
**Про процесс:**

Весь проект сделан нейросетями. [Что именно] писала нейросеть, [что] делала нейросеть.
Я только говорил что хочу — и смотрел что получается.

И этот пост тоже написала нейросеть. И постит его робот, прикрученный к каналу.

Мы живём в будущем 🤖
```

## Отправка через бота (PowerShell)

```powershell
$botToken = '1234567890:AABBCCxxx'
$chatId   = '@channel_name'  # или числовой ID канала

# Только текст
$body = @{
    chat_id    = $chatId
    text       = $text
    parse_mode = 'Markdown'
}
Invoke-RestMethod "https://api.telegram.org/bot$botToken/sendMessage" -Method POST -Body $body

# Фото из файла + caption (≤1024)
$form = @{
    chat_id    = $chatId
    caption    = $caption
    parse_mode = 'Markdown'
    photo      = Get-Item "path\to\image.png"
}
Invoke-RestMethod "https://api.telegram.org/bot$botToken/sendPhoto" -Method POST -Form $form
```

## Получить chat_id канала
```powershell
# Отправить любое сообщение боту в канале, потом:
Invoke-RestMethod "https://api.telegram.org/bot$botToken/getUpdates"
# Найти "chat":{"id": -100XXXXXXXXXX} — это и есть chat_id
```
